Ma configuration LaTeX dans le Cloud

Date Ven 03 décembre 2010 Tags LaTeX / configuration / Cloud

Pour partager la configuration LaTeX entre plusieurs ordinateurs j'ai placé mon dossier texmf avec mes personalisations dans le "Cloud", en fait dans un dossier synchronisé avec "Copy" mais ça marche aussi bien avec "DropBox" que j'utilise aussi mais pour d'autres choses.

Le chemin de ce dossier est : /Users/marc/Cloud/Copy/latex/texmf/tex/ et il doit être organisé avec un arbre TeX pour être reconnu à la compilation (Dossiers tex, latex, bibtex/bst et bibtex/bib ).

et pour qu'il soit reconnu par LaTeX j'ai édité le fichier: /usr/local/texlive/2013/texmf.cnf pour le désigner à la variable TEXMFHOME.

Cela donne:

% (Public domain.)
% This texmf.cnf file should contain only your personal changes from the
% original texmf.cnf (for example, as chosen in the installer).
%
% That is, if you need to make changes to texmf.cnf, put your custom
% settings in this file, which is .../texlive/YYYY/texmf.cnf, rather than
% the distributed file (which is .../texlive/YYYY/texmf-dist/web2c/texmf.cnf).
% And include *only* your changed values, not a copy of the whole thing!
%
TEXMFHOME = ~/Cloud/Copy/latex/texmf
TEXMFVAR = ~/Library/texlive/2013/texmf-var
TEXMFCONFIG = ~/Library/texlive/2013/texmf-config

Vous pouvez vérifier que la variable est bien prise en compte avec la commande:

kpsewhich -var-value TEXMFHOME

ou encore plus généralement:

texconfig conf

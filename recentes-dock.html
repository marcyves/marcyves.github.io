Les applications récentes dans le Dock

Certes avec un Macbook on peut retrouver une application en 2 secondes grâce au spotlight. On peut aussi mettre ses applications favorites dans le Dock, mais il est toujours utile d’avoir un listing des dernières applications utilisées ou encore des documents récemment ouverts. Pour se faire, c’est simple :

defaults write com.apple.dock persistent-others -array-add '{ "tile-data" = { "list-type" = 1; }; "tile-type" = "recents-tile"; }'

Relancez le Dock en tapant : killall Dock

Si vous souhaitez ajouter un effet au hover des icones, tapez ceci : defaults write com.apple.dock mouse-over-hilte-stack -boolean YES
